# Bachelor Landing Page

Односторінковий вебсайт для представлення бакалаврської роботи на тему:

**«Моделювання процесу зближення та стикування космічних апаратів»**

## Реалізовано
- назва роботи українською та англійською
- короткий опис
- ключові слова
- мета дослідження
- основні завдання
- методологія
- очікувані результати
- контактна інформація

## Технології
- HTML5
- CSS3
